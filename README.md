## 🛠 Tech Stack

- **Framework:** SwiftUI
- **Architecture:** Clean Architecture
- **Reactive Programming:** Combine
- **Minimum iOS Version:** iOS 16.0+
- **Xcode Version:** 15.4+

## 📚 Libraries

- **Firebase iOS SDK**
  - Authentication
  - Firestore
  - RemoteConfig
 
- **Amplitude**

## 🌟 Key Features

- 실시간 좌석 현황 확인
- 이용 시간 관리
- 알림 서비스
- 학과별 랭킹

## 🏗 Architecture

본 프로젝트는 Clean Architecture를 기반으로 구현되었으며, 다음과 같은 레이어로 구성되어 있습니다:

- **Presentation Layer (SwiftUI Views)**
  - SwiftUI View

- **Domain Layer**
  - Interactors
  - Entities
  - Repository Interfaces

- **Data Layer**
  - Repository Implementations
  - Data Sources
  - DTOs
